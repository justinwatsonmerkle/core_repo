# Career OS Commands

Use these commands in chat to control output.

- `CHECKPOINT`
  - Save progress mid-session.
  - Assistant produces a minimal HARVEST + COMMIT PACK as a single zip of updated files.

- `WRAP_UP`
  - End the session and publish changes.
  - Assistant produces full HARVEST + COMMIT PACK as a single zip of updated files.

- `CARRY_OVER`
  - If you must move to a new chat mid-session, ask the assistant to produce a short carry-over note
    (5â€“10 lines) describing what was learned and what is next.
